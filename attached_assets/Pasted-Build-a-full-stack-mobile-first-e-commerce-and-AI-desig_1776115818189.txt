Build a full-stack, mobile-first e-commerce and AI design platform for **Signitive Enterprises** — a premium custom apparel manufacturer based in Pakistan. The platform allows customers to design, preview, and order custom-printed/embroidered clothing using an AI Design Studio.

---

## BRAND IDENTITY
- **Brand:** Signitive Enterprises
- **Aesthetic:** Premium, high-contrast, black and gold — clean, minimalist, bold typography
- **Reference site for UI/UX inspiration:** https://7thearthstudios.com (clean layout, full-bleed imagery, luxury feel)
- **Logo:** Use "SIGNITIVE ENTERPRISES" wordmark with lightning bolt motif (black/gold palette)
- **Website:** www.signitiveenterprises.com

---

## TECH STACK
- **Frontend:** Next.js 14 (App Router) + Tailwind CSS
- **Backend:** Node.js / Next.js API routes
- **Database:** Supabase (PostgreSQL)
- **AI Image Generation:** Replicate API (use Stable Diffusion XL or FLUX for prompt-to-design)
- **Image Mapping / Mockup:** Fabric.js or Konva.js for canvas-based design placement on product mockups
- **Auth:** Supabase Auth (email/password + Google OAuth)
- **Payments:** Stripe (with PKR support for local; USD for international)
- **Storage:** Supabase Storage (for user-uploaded logos and generated designs)
- **Hosting Target:** Vercel-compatible structure

## CORE PAGES & FEATURES

### 2. The AI Design Studio (/studio) — CORE FEATURE
This is the main workspace. Build it as a split-panel interface:

**Left Panel — Product Selector**
- Dropdown or visual card picker for product category and sub-category (see catalog below)
- Color picker for base garment color
- Size selector
- Fabric type selector: 100% Cotton / Polyester Blend / Cotton-Poly Mix
- GSM weight selector: 160gsm / 180gsm / 220gsm / 280gsm
- Option: "Add Your Own Brand Label" (checkbox — neck label / hem label)

**Center Panel — Live Canvas Preview**
- Display a flat-lay or ghost mannequin mockup of the selected product
- Use Fabric.js canvas overlaid on the product image
- The design (AI-generated or uploaded) renders on the product with basic displacement/warp effect to follow fabric folds
- Multi-angle toggle: Front / Back / Left Sleeve / Right Sleeve
- Zoom in/out controls

**Right Panel — Design Input**
- **Tab 1: Prompt-to-Design (AI Generate)**
  - Large text input: "Describe your design..." (e.g., "A vintage 90s eagle with neon lightning bolts")
  - Style modifiers: [Vintage] [Minimalist] [Bold] [Streetwear] [Luxury] [Grunge] toggles
  - Color palette picker
  - "Generate Design" button → calls Replicate API → renders result on canvas
  - Regenerate / Variations buttons
- **Tab 2: Upload Your Logo**
  - Drag-and-drop or file upload (PNG, SVG, AI files)
  - Auto-removes background (use remove.bg API or Replicate)
  - Places logo on canvas; user can resize/reposition
- **Tab 3: Text Add-On**
  - Add custom text, choose font (Google Fonts), size, color, placement

**Bottom Bar (Studio)**
- "Save Design" (saves to user account)
- "Add to Quote" → goes to bulk order flow

### 3. Bulk Order Quote Engine (/quote)
- Table showing: Product | Quantity | Unit Price | Discount | Total
- Auto-calculate tiered discounts:
  - 1–49 units → base price
  - 50–99 units → 10% off
  - 100–199 units → 20% off
  - 200+ units → 25% off + "Contact for custom pricing" flag
- **MOQ Progress Bar:** Visual bar showing how many units until next discount tier
- Production time estimate based on quantity
- "Request Quote" button → sends email via Resend/Nodemailer to info@signitiveenterprises.com + customer confirmation

## UI/UX DESIGN SYSTEM
- **Background:** #0a0a0a (near black)
- **Primary Gold:** #C9A84C
- **Text:** #FFFFFF primary, #A0A0A0 secondary
- **Accent:** #D4AF37 (gold shimmer for CTAs)
- **Font:** Inter or Satoshi for body; Bebas Neue or Barlow Condensed for display headings
- **Cards:** Dark glass-morphism style (#1a1a1a with subtle gold border)
- **Buttons:** Solid gold (#C9A84C, black text) for primary; outlined gold for secondary
- **Hover states:** Subtle gold glow / shimmer effect
- No rounded corners on product cards (sharp edges = premium/streetwear feel)
